# fuzzWeb
___

web fuzzer for penetration testing and bruteforcing

| Name              | Usage                                           |
| ---               | ---                                             |
| --url, -u         | Enter url to be bruteforced                     |
| --wordlist, -w    | The wordlist to be used for bruteforcing        |
